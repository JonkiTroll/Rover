# Rover
Implementation of a rc tank. Based on rctestflight's design.

# Requirements

pyttsx3 python library for RoverTTS.py. For Ubuntu linux, libespeak1 needs to be installed manually (sudo apt install libespeak1).

OpenCv2 for Video_server.py. 

# Utilities
GPS 	GY-NEO6MV2

IMU	Icm20948 (Pid controller)

Camera (Servo)  logitech webcam

Speakers

Temp, gas, barometer, humidity https://www.bosch-sensortec.com/products/environmental-sensors/gas-sensors/bme688/

Neopixels

Battery management (Fuel gauge) https://www.ti.com/product/BQ34110#product-details##description

expansion headers (Servos, analog pins)

Hall Effect sensors x2 (3390Q)
